# Shadow Leap Demo

Landing page game demo su dung HTML, CSS va JavaScript (Three.js) de hien thi scene 3D va nhan vat FBX.

## Chay du an

1. Mo thu muc du an trong VS Code.
2. Chay Live Server voi `index.html`.
3. Mo trinh duyet tai dia chi local duoc cung cap (vi du: `http://127.0.0.1:5500/index.html`).

## Cau truc chinh

- `index.html`: layout trang.
- `style.css`: giao dien va responsive.
- `main.js`: logic Three.js, load model FBX, animation va interaction.
- `Jump.fbx`: model nhan vat.

## Ghi chu

- Neu nhan vat hien thi toi/mat mau, kiem tra file model va texture di kem.
- Nen dung dinh dang GLB de mau on dinh hon tren web.
